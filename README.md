## 🚀 About Me

I'm a Computer Science student at **KL University** with a passion for **Full-Stack Development**, **REST API design**, and **Data Analytics**. I build scalable, secure web applications using **React.js, Spring Boot, Java, and MySQL**, and I love turning complex problems into clean, working software.

- 🎓 B.Tech CSE @ KL University — CGPA **9.21/10**
- 🛠️ Currently building full-stack apps with **Spring Boot + React.js**
- 🔐 Passionate about **Cybersecurity** and secure software design
- 📊 Exploring **Power BI** and data analytics on the side
- 🌱 Certified in **MongoDB DBA** and **Azure Fundamentals (AZ-900)**
